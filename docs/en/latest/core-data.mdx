## Summary
World lifecycle and storage systems used by HytaleServer.

## Universe
- Universe coordinates worlds and readiness
- Worlds are managed under server.core.universe.world

## Storage
- DataStoreProvider defaults to disk storage
- Player storage uses PlayerStorageProvider
- Storage classes use Zstd for compression
