## Кратко
Как плагины обнаруживаются, загружаются и управляются ядром сервера.

## Обнаружение
- Core-плагины регистрируются при запуске
- Внешние плагины грузятся из mods/ и доп. директорий
- Builtin плагины могут лежать рядом с сервером

## Жизненный цикл
- PluginManager выполняет фазы setup и start
- Порядок загрузки решается по зависимостям
- Состояние плагина хранится в PluginState

## Classloading
- Core-плагины используют общий PluginClassLoader
- Внешние плагины изолированы по пути
- Early-плагины могут трансформировать классы до boot
