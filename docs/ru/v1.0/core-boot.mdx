## Кратко
Последовательность запуска, события жизненного цикла и shutdown HytaleServer.

> Примечание о версии: это зафиксированный снимок v1.0. Обновления идут в latest.

## Запуск
- Main загружает early-плагины и может включить transforming classloader
- LateMain настраивает логирование, читает CLI и создает HytaleServer
- HytaleServer.boot поднимает команды, плагины, ассеты и universe

## События
- LoadAssetEvent валидирует ассеты
- BootEvent отправляется после готовности universe
- ShutdownEvent отправляется при завершении

## Shutdown
- Причины shutdown отслеживаются через ShutdownReason
- Плагины останавливаются до закрытия сетевых слушателей
- Конфиг сохраняется по расписанию
