## Кратко
Жизненный цикл мира и системы хранения HytaleServer.

> Примечание о версии: это зафиксированный снимок v1.0. Обновления идут в latest.

## Universe
- Universe управляет мирами и готовностью
- Миры находятся в server.core.universe.world

## Хранилище
- DataStoreProvider по умолчанию использует диск
- PlayerStorageProvider отвечает за данные игроков
- Классы хранения используют Zstd

## Связанные пакеты
- server.core.universe.datastore
- server.core.universe.playerdata
- storage (IndexedStorageFile)
