## Кратко
Конфигурация сервера и ключевые параметры запуска.

> Примечание о версии: это зафиксированный снимок v1.0. Обновления идут в latest.

## Config
- config.json загружается через HytaleServerConfig
- Поля: serverName, motd, maxPlayers, maxViewRadius
- modules и modConfig управляют фичами

## Сеть
- connectionTimeouts управляет таймаутами
- rateLimitConfig включает лимиты пакетов

## CLI флаги
- --bind, --transport, --auth-mode
- --assets, --mods, --universe
- --validate-assets, --shutdown-after-validate
- --event-debug, --bare, --disable-sentry
